# Cheat Sheet

Cumulative key takeaways — one section per lesson. Newest at the bottom.

## 2026-06-20 — Lesson 1: Self-introduction & small talk (goal: articles a/an/the)

### 词汇 / 表达 Expressions
- **I used to work at...** — 讲过去的工作经历. e.g. I used to work at Baidu for three years.
- **work mainly on...** — 简洁描述工作重心. e.g. I work in the platform team, mainly on payment systems.
- **Let's grab coffee sometime** — 友好地约同事见面.

### 短语 / 缓冲语 Phrases (incl. filler)
- **Let me think for a second… / That's a good question…** — 卡壳时争取时间，别冷场或转中文.

### 语法 Grammar
- **冠词 a/an/the**: `an` + 元音音（an RD, an honest engineer）; "我自己的"用 my 不用 the（improve my ability）; go to **the** office, in **the** morning.
- **"work mainly on X"**: 骨架是 "work on X"（负责某方面）; "on X" 作状语修饰 work; "mainly" 把范围收窄成"主要是".
- **过去式讲故事**（下节课重点）: 全程过去式 find→found, choose→chose, go→went; leave 不加 from（leave Baidu）; this/last/next week 前不加 in.

### 发音 Pronunciation
- **an + 元音音**: an RD ("ar-dee"), an honest (h 不发音).
- **choose /tʃuːz/ → chose /tʃoʊz/**（过去式发音也变）.

### 易错清单 Error checklist
- a RD → **an** RD
- responsible for develop → responsible for **developing**
- success project → a **successful** project
- improve the ability → improve **my** ability
- I choosed → I **chose**
- leave from Baidu → leave Baidu
- in this week → this week
