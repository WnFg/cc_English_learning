## Wrap-up

### 🧠 今日核心总结 (Key takeaways)
- **词汇 Vocabulary:** coincidence (巧合), have sth. in common (有共同点), run into a problem → ran into (遇到问题)
- **短语 Phrases (incl. filler):** "it turned out (that)..." / "in the end, I managed to..." / "Let me think for a second…" (filler when stuck)
- **语法 Grammar:** Past tense consistency — once you start a past story, every verb follows: stopped, checked, ran, turned out, managed. No switching to present mid-story.
- **发音 Pronunciation:** "turned" = /tɜːrnd/ (don't drop the -d); "managed" = MAN-aged (stress on first syllable)

### ❗ Key errors
- "the SSO token mismatches" → "didn't match" (tense switch mid-story)
- "lot's of tests" → "a lot of tests" (no apostrophe; "a lot of" is one phrase)
- "replace X to Y" → "replace X with Y" (fixed collocation)
- From homework: "this work is very boring" → "the work was boring" (past tense consistency)
- From homework: "I leave Baidu" → "I left Baidu" (irregular past)
- From homework: "an another" → "another"
